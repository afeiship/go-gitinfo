# sso-weibo-node
> Weibo sso based on node.

## installation
```shell
npm i @feizheng/sso-weibo-node
```

## usage
```js
import weiboSso from '@feizheng/sso-weibo-node';

weiboSso({ username: 'YOUR_USERNAME', password:'YOUR_PASSWORD' }).then(resp => {
  // GET THE SUB_COOKIE
});
```
